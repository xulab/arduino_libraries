1.首先安装attiny44的arduino ide插件，安装方法见：http://highlowtech.org/?p=1695，或者为之笔记的“High-Low Tech C Programming an ATtiny w/ Arduino 1.6 (or 1.0)”

2. board选择：Attiny
processor选择：Attiny44
clock选择：8MHz（internal）
programmer 选择usbtinyisp

3. Programmer 选择 usbtinyisp

4. 选择Burn Boot Loader

5.打开程序选择file->upload use programmer
